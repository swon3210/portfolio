Mybrary

Node.js

기획 의도

Mybrary는 이용자 스스로가 본인이 현재까지 읽은 도서, 블로그 글, 유튜브 영상 등을 분류하고 사람들에게 추천해주는 '사서'가 되어, 본인만의 도서관을 인터넷 상에 개설할 수 있게 해주는 서비스입니다. 여러 대중적인 서비스의 API 를 활용하여 본인의 자료 데이터베이스인 '서가'를 제작하고, 각 자료를 설명해주는 '포스트'를 작성하는 기능을 기본으로 지닙니다.

프로젝트 내의 역할

기획, 디자인, 개발과 운영환경 구축 모두 제가 담당하는 개인 프로젝트입니다.

개발 과정

백엔드 REST API 서버 개발 / API 명세 작성

Node.js 를 Sequelize ORM과 함께 서버사이드 개발을 위해 사용하였고 MySQL로 데이터베이스를 구축했습니다. Gitbook을 통해 API 명세를 문서화하였습니다.
