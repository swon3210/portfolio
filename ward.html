WARD SHOP

Client : Vue.js / Admin : React.js

기획 의도

WARD SHOP은 스터디 송별회 떄, 장난스럽게 이야기가 나온 프로그래머 용품 쇼핑몰 아이디어를 실제로 구현해본 웹 사이트입니다. 물품의 조회, 검색, 구매, 구매 정보 조회와 같은 고객을 위한 클라이언트 웹사이트는 물론이고, 쇼핑몰의 관리자가 직접 상품을 추가하고, 상품 정보를 관리하고, 카테고리와 카테고리 분류를 수정하고, 전체 판매 현황을 조회해볼 수 있는 관리자 페이지까지 구현시키는 것을 목표로 제작되었습니다.

프로젝트 내의 역할

Adobe XD를 이용한 UI/UX 프로토타이핑과 UI 컴포넌트 문서화, 로그인 페이지, 상품 조회 페이지, 상품 상세 설명 페이지의 프론트엔드 작업을 맡았습니다.

개발 과정

UI 컴포넌트 제작 및 문서화

UI 컴포넌트 문서화 툴인 StoryBook 를 이용하여 UI 컴포넌트 명세 사이트를 만들어 정적 배포하여 다른 프론트엔드 개발자들도 저의 Vue.js, React.js 컴포넌트를 이용하는데 있어서 편의를 얻을 수 있도록 하였습니다. React.js 컴포넌트들은 npm 라이브러리로 배포하였습니다.

Vue.js 컴포넌트 Storybook react-uikit-ward NPM 패키지 React.js 컴포넌트 Storybook

퍼블리싱/프론트엔드

프로젝트는 프론트엔드 서버와 백엔드 서버를 분리하여 개발되었고, Vue.js 를 통해 개발한 웹 어플리케이션 또한 SEO 처리가 가능하도록 하기 위해 Nuxt 프레임워크를 통해 정적 파일로 빌드시켜 개발하였습니다.
